Sobre o Projeto

A Academia Internacional de Ciências e Artes é uma iniciativa acadêmica voltada à integração entre tecnologia, arte e conhecimento científico. Este projeto foi desenvolvido utilizando Python e Flask, com o objetivo de criar uma plataforma interativa que conecta estudantes, pesquisadores e artistas ao redor do mundo.
